# Git Practical Examination

## Overview
This repository is created for the Git practical examination.
Students are required to demonstrate their understanding of Git
version control concepts through hands-on tasks.

The exam focuses on repository initialization, branching, committing,
merging, conflict resolution, and commit history manipulation.
